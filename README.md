# MySkills

This repository contains reusable Codex/agent skills for engineering work.

## Included skills

- `skills/bha-s-engineering-skill-v1/` — **BHA-S Engineering Skill V1.0**, a scalable bionic homeostasis architecture skill for software, algorithm, AI-tooling, and robotics engineering projects.

## How to use with Codex

Point Codex at this repository or copy the desired skill folder into the skills location supported by your Codex environment. The mandatory entry file is `SKILL.md`; supporting material is under `docs/`, `templates/`, and `tools/`.
